1. What is PushForever ?

PushForever can help send push notification from our api to your device which has already available Google Play and App Store
You can see your token and user key in your control panel. Token and user key will not change.

2. How can i use ?
POST an Http request http://api.pushforever.online/api/v1/push with the following parameters
  • token Your application token key which you can get from member area
  • user Your user key which you can get from member area
  • title
  • message
